# NIOS Kids Learning Platform

A child-friendly, NIOS-aligned learning website designed to support families with flexible learning pathways, clear parent guidance, and structured mentorship.

## Features
- Program overview for different learning stages (Foundation to Senior Secondary)
- Learning journey pages to help parents understand the roadmap
- Gallery and activities to make learning feel engaging
- Counselling/contact section for guidance
- FAQ and admissions information

## Tech Stack
- Next.js (React) + TypeScript
- Tailwind CSS
- Framer Motion (animations)

## Getting Started
1. Install dependencies:
   - `npm install`
2. Run the development server:
   - `npm run dev`
3. Open http://localhost:3000

## Scripts
- `npm run dev` — start dev server
- `npm run build` — build for production
- `npm run lint` — lint the codebase
